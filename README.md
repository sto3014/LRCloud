## Building and Installing

Ensure you have CMake installed before proceeding.

### Steps:
1. Clone the repository:
   ```sh
   git clone <repo-url>
   cd <repo-name>
   ```
2. Run the build script:
   ```sh
   chmod +x build.sh
   ./build.sh
   ```


## Requirements
- macOS
- CMake
- Clang (default on macOS)
